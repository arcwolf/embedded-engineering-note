# 임베디드 시스템 공학 노트

임베디드 시스템 개발을 위한 공학 지식을 정리한 노트입니다.

---

## 목차

### 1. 회로 이론 (Circuit Theory)
- 1.1 기본 회로 법칙 (옴의 법칙, KVL, KCL)
- 1.2 교류/직류 회로 해석
- 1.3 임피던스와 리액턴스
- 1.4 공진 회로
- 1.5 필터 회로

### 2. 아날로그 회로 (Analog Circuits)
- 2.1 연산 증폭기 (Op-Amp)
- 2.2 증폭기 회로
- 2.3 발진기 회로
- 2.4 전원 회로 (LDO, SMPS)
- 2.5 신호 조절 회로

### 3. 디지털 회로 (Digital Circuits)
- 3.1 논리 게이트 및 불 대수
- 3.2 조합 논리 회로
- 3.3 순서 논리 회로 (플립플롭, 레지스터)
- 3.4 FSM (유한 상태 기계)
- 3.5 타이밍 분석

### 4. 반도체 소자 (Semiconductor Devices)
- 4.1 다이오드
- 4.2 트랜지스터 (BJT, MOSFET)
- 4.3 사이리스터 (SCR, TRIAC)
- 4.4 집적 회로 (IC)
- 4.5 센서 소자

### 5. 부품 및 인터페이스 (Components & Interface)
- 5.1 수동 소자 (저항, 커패시터, 인덕터)
- 5.2 커넥터 및 통신 인터페이스
- 5.3 디스플레이 및 입출력 장치
- 5.4 전원 관리 IC
- 5.5 통신 모듈 (Wi-Fi, BT, LoRa 등)

### 6. 제어 이론 (Control Theory)
- 6.1 시스템 모델링 (전달 함수, 상태 방정식)
- 6.2 PID 제어
- 6.3 안정성 해석 (보드 선도, 나이퀴스트)
- 6.4 디지털 제어
- 6.5 모터 제어

### 7. 임베디드 SW 개발 (Embedded Software)
- 7.1 마이크로컨트롤러 아키텍처 (ARM Cortex-M 등)
- 7.2 펌웨어 개발 (C/C++)
- 7.3 RTOS (FreeRTOS 등)
- 7.4 주변장치 드라이버 (GPIO, UART, SPI, I2C, ADC)
- 7.5 부트로더 및 플래시 메모리
- 7.6 디버깅 및 테스트 (JTAG, SWD)

### 8. 통신 프로토콜 (Communication Protocols)
- 8.1 직렬 통신 (UART, RS-232, RS-485)
- 8.2 동기 직렬 통신 (SPI, I2C)
- 8.3 필드버스 (CAN, LIN)
- 8.4 이더넷 및 TCP/IP
- 8.5 무선 통신

### 9. PCB 설계 (PCB Design)
- 9.1 회로도 작성
- 9.2 PCB 레이아웃 규칙
- 9.3 신호 무결성 (SI)
- 9.4 전원 무결성 (PI)
- 9.5 EMC/EMI 설계
